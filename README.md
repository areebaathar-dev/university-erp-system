# University ERP System

## Overview
A web-based University ERP system designed to manage academic and administrative operations efficiently.

## Features
- Role-based login (Admin, Faculty, Student)
- Student attendance management
- Marks and grading system
- Fee record management
- Centralized database system
- Reporting modules

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- Bootstrap

## Database Design
Structured relational database with multiple tables for users, attendance, marks, and fees.

## Purpose
Developed as an academic project to understand enterprise-level system design and database management.
